# [step1] 유효성 검사 및 예외처리

**과제 요구 사항**: 상품명 입력에 대한 예외처리
- 공백 포함 최대 **15자**
- 특수문자: `()`, `[]`, `+`, `-`, `&`, `/`, `_` 만 사용 가능
- `"카카오"`가 포함된 문구는 담당 MD와 협의한 경우에만 사용 가능
