### 회원 관리 시스템
- [ ] 회원 가입 기능
  - [ ] 이메일과 비밀번호를 통한 회원 가입
  - [ ] 회원 가입 시 JWT 토큰 발급
  - [ ] 중복 이메일 가입 방지
- [ ] 로그인 기능
  - [ ] 이메일과 비밀번호를 통한 로그인
  - [ ] 로그인 성공 시 JWT 토큰 발급
  - [ ] 잘못된 로그인 정보 처리 (403 Forbidden)

  - [ ] 201 Created (회원 가입 성공)

  - [ ] 회원 조회 기능
  - [ ] 회원 추가 기능
