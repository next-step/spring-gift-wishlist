- 이름 유효성 검사 조건
  1. 상품 이름은 공백을 포함하여 최대 15자까지 입력할 수 있다.
  2. 특수 문자는 (),[],+,-,&,/,_ 를 제외하고 사용할 수 없다
  3. "카카오"가 포함된 문구는 담당 MD와 협의한 경우에만 사용할 수 있다.
