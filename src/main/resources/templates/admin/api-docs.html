API 개요

기본 정보

  • Base URL: http://localhost:8080/api
  • Content-Type: application/json
  • 인증: JWT Bearer Token

응답 형식

  • 성공: HTTP 200-299
  • 클라이언트 오류: HTTP 400-499
  • 서버 오류: HTTP 500-599

인증 API

로그인

POST /api/auth/login

사용자 로그인 및 JWT 토큰 발급

Request Body:

{
  "email": "user@example.com",
  "password": "password123"
}

회원가입

POST /api/auth/signup

새 사용자 등록

Request Body:

{
  "email": "user@example.com",
  "password": "password123",
  "name": "홍길동"
}

상품 API

상품 목록 조회

GET /api/products

페이지네이션된 상품 목록 조회

Query: page, size

Response:

{
  "contents": [...],
  "page": 0,
  "size": 10,
  "totalPages": 5,
  "totalElements": 50
}

상품 상세 조회

GET /api/products/{id}

특정 상품의 상세 정보 조회

Response:

{
  "id": 1,
  "name": "상품명",
  "price": 10000,
  "imageUrl": "https://...",
  "description": "상품 설명"
}

사용자 API

사용자 프로필 조회

GET /api/users/profile

인증된 사용자의 프로필 정보 조회

인증 필요

Response:

{
  "id": 1,
  "email": "user@example.com",
  "name": "홍길동",
  "createdAt": "2024-01-01T00:00:00"
}
